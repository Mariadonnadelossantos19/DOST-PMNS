POSTMAN TEST FOR SUPER ADMIN LOGIN
=====================================

1. METHOD: POST
2. URL: http://localhost:4000/api/auth/login
3. HEADERS:
   Content-Type: application/json

4. BODY (raw JSON):
{
  "email": "admin@dost-mimaropa.gov.ph",
  "password": "admin123"
}

5. EXPECTED RESPONSE:
{
  "success": true,
  "message": "Login successful",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "_id": "64f8a1b2c3d4e5f6a7b8c9d0",
    "userId": "SUPER_ADMIN_001",
    "firstName": "Super",
    "lastName": "Administrator",
    "email": "admin@dost-mimaropa.gov.ph",
    "role": "super_admin",
    "department": "DOST MIMAROPA",
    "position": "Regional Director",
    "status": "active"
  }
}

6. SAVE THE TOKEN:
   - Copy the "token" value from the response
   - Use it in Authorization header for other requests:
   - Header: Authorization
   - Value: Bearer YOUR_TOKEN_HERE

7. TEST OTHER ENDPOINTS:
   - GET /api/users (with Authorization header)
   - GET /api/programs (with Authorization header)
   - GET /api/pstos (with Authorization header)
   - GET /api/proponent-requests (with Authorization header)
   - GET /api/enrollments (with Authorization header)
